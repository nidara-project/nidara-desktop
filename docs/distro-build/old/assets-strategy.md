# Estrategia de Activos Visuales (Assets)\n\nPara diferenciar Crystal Shell, reemplazamos los activos por defecto de Ubuntu por opciones \"Squircle\" modernas.\n\n## 1. Iconos: Reversal Theme\nHemos seleccionado **Reversal** por su equilibrio entre la forma de Apple y un diseño original.\n- **Repositorio**: `https://github.com/yeyushengfan258/Reversal-icon-theme`\n- **Instalación Automatizada**:\n  ```bash\n  git clone https://github.com/yeyushengfan258/Reversal-icon-theme.git\n  cd Reversal-icon-theme\n  ./install.sh -a -d ~/.local/share/icons  # Instalar todas las variantes en usuario\n  ```\n- **Configuración GTK**:\n  `gsettings set org.gnome.desktop.interface icon-theme 'Reversal-dark'`\n\n## 2. Cursores\nRecomendamos **Bibata Modern Ice** para un cursor suave y moderno que complementa la estética glass.\n\n## 3. Fuentes (Tipografía)\n- **UI**: Inter (o SF Pro si la licencia lo permite, cuidado con la ISO pública).\n- **Mono**: JetBrains Mono o Geist Mono.\n
